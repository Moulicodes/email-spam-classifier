# Email Spam Classification using Machine Learning

An end-to-end Machine Learning pipeline built in Python to classify emails as Spam or Ham using TF-IDF feature extraction and Logistic Regression.

## Dataset
The dataset used in this project is the **Spam Email Dataset** sourced from Kaggle. 
* **Source Link:** [Kaggle Spam Email Dataset](https://www.kaggle.com/datasets/abdmental01/email-spam-dedection)
* **File Name:** `mail_data.csv`

> **Note:** To run this project locally or in Google Colab, please download the `mail_data.csv` file from the link above and upload it to your active environment directory.

## Performance
* **Training Accuracy:** 96.77%
* **Testing Accuracy:** 96.68%
